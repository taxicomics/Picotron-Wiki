# deli(table,[index]): any

## Overview

`deli` deletes the element at index `[index]` in the table `table`, defaulting to the last element of the table.

## Arguments

### `table`: table

The table you would like to delete from.

### `[index]`: integer

The index of the element to delete from the table `table`.

## Returns

### element: any

Returns the element that was deleted
