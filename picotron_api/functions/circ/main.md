# circ(x,y,r,[col])

## Overview

`circ` draws a circle at `x`,`y` with radius `r`.

If `r` is negative, the circle is not drawn.

For drawing a filled circle, see [`circfill()`](/picotron_api/functions/circfill/main.md).

## Arguments

### `x`: number

The x-coordinate of the circle

### `y`: number

The y-coordinate of the circle

### `r`: number

The radius of the circle

### `[col]`: number

The color of the circle; defaults to the last used color.
